# Мини-опросник на React

Это небольшое React-приложение — мини-викторина с тремя вопросами. Пользователь выбирает ответы, а в конце видит количество правильных ответов.

## Описание

Приложение демонстрирует умение работать с React-компонентами, состоянием (`useState`), обработчиками событий и условным рендерингом.  

## Основные возможности

1)Показ вопросов один за другим
2)Варианты ответов в виде кнопок
3)Подсчёт правильных ответов
4)Отображение результата по завершении
